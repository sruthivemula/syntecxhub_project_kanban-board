# 📋 SmartFlow Kanban Board

A modern and responsive Kanban Board web application built using **HTML, CSS, and JavaScript**. This project helps users organize tasks efficiently with a clean dashboard interface, drag-and-drop functionality, and task management features.

---

## 🚀 Features

* ✅ Add New Tasks
* ✅ Delete Tasks
* ✅ Drag & Drop Functionality
* ✅ Search Tasks
* ✅ Dark/Light Mode
* ✅ Task Statistics Dashboard
* ✅ Responsive Design
* ✅ Local Storage Support
* ✅ Modern UI/UX Design
* ✅ Real-Time Task Counter

---

## 🛠️ Technologies Used

## 🎯 How to Use

1. Open the application in your browser.
2. Enter a task in the input field.
3. Click **Add Task**.
4. Drag tasks between:

   * To Do
   * Doing
   * Done
5. Use the search bar to find tasks quickly.
6. Toggle between Dark and Light mode.
7. Your tasks are automatically saved using Local Storage.
